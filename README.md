# Portfólio Pessoal

Projeto de portfólio pessoal criado com HTML e CSS. O site apresenta uma página única com navegação interna, seção de destaque, projetos, habilidades e área "Sobre mim".

## 📌 Visão geral

- **Título do site:** Portifolio
- **Tecnologias:** HTML5, CSS3
- **Arquitetura:** CSS modular por componentes
- **Imagens:** arquivo `image/` contém ícones, projetos, status e elementos visuais

## 🧩 Estrutura do projeto

```
templateSite2/
├── index.html              # Página principal
├── README.md               # Documentação
├── css/
│   ├── reset.css           # Normalização de estilos
│   ├── style.css           # Estilos globais e variáveis
│   ├── variable.css        # Estilo de fontes
│   └── components/
│       ├── header.css      # Estilos do header e navegação
│       ├── hero.css        # Estilos do hero
│       ├── projects.css    # Estilos da parte de Project
│       ├── projectcard.css # Estilos dos cards de projeto
│       └── footer.css      # Estilos do footer
└── image/
    ├── favicon/            # Ícone do site
    ├── hero/               # Imagem do hero
    ├── logo/               # Logo
    └── projects/           # Imagens dos projetos
```

## ✨ Funcionalidades

- Navegação interna via menu fixo
- Seção de destaque com apresentação pessoal
- Galeria de projetos com filtros visuais
- Layout responsivo e organizado por componentes
